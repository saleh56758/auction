# Secret Auction Program

A Python command-line application to simulate a blind auction. Bidders enter their names and bids privately, and the program determines the highest bidder.

## Features

* **Private Bidding:** Uses a pseudo-clear screen function to hide previous bids.
* **Winner Calculation:** Automatically identifies the highest bidder and their bid amount.

## How to Run

1. Ensure you have Python installed.
2. Run the script from your terminal:
```bash
python auction.py

```


3. Follow the on-screen prompts to enter names and bid amounts.
